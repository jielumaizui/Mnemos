**发现并修复的问题：**
1. `save_to_memos` 缺少 `layer=L1` 标签 → `fetch_l1_sessions` 无法识别 → **修复**：添加 `layer=L1`
2. `save_to_memos` 缺少 `session` 标签 → doc 无法被分组为 session → **修复**：添加 `session=doc-{hash}`
3. `distill_all.py` `len(messages) >= 5` 门槛 → doc session 只有 1 条消息被 SKIP → **修复**：doc session 使用 `min_messages=1`
4. `engine.process()` 对 doc session 返回 `judgment=skip`（设计用于对话）→ **修复**：doc session 不走 LLM，直接解析生成 wiki
5. `generate_wiki_page` 清理逻辑太激进，清除了所有 `type` 键 → **修复**：只清除值在 `FORM_TO_ENTITY_TYPE` 键中的旧类型名
6. `mnemos_daemon.py` Memos 扫描逻辑同样跳过 doc session → **修复**：daemon 直接处理 doc session，不入队 amphora

**新增函数：** `process_doc_session()` in `distillation_engine.py` — 外部文档直出 wiki，无需 LLM。
